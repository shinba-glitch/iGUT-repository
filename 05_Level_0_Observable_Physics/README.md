## レベル0 観測可能な物理と宇宙の歴史の層

レベル0は、レベル1で創発した時空と物理法則が  
観測可能な現象として展開される層である。  
この層では、量子場、素粒子、宇宙の歴史、構造形成、  
ブラックホール、重力波、宇宙背景放射など、  
実際に観測される物理現象がすべて記述される。

レベル0は、iGUTの全階層の中で  
「理論が現実と接続する層」であり、  
宇宙の歴史と観測可能な物理が統一的に理解される場所である。

## レベル0の目的

レベル0の目的は、レベル1で定まった  
- 時空  
- 量子  
- 重力  
- ゲージ対称性  
- 物質  
- 物理定数  

が、どのようにして  
観測可能な現象として展開されるかを記述することである。

この層では、宇宙の歴史が  
情報の流れとして理解され、  
観測可能な物理がその投影として現れる。

レベル0には、次の四つの領域が存在する。

- 初期宇宙と構造形成

$$
